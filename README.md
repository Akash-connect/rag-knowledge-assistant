# RAG-Based LLM Knowledge Assistant

A Retrieval-Augmented Generation (RAG) AI assistant built with:

- LangChain
- ChromaDB
- Hugging Face Embeddings
- Groq LLM
- Streamlit

## Features

- Upload PDF/TXT documents
- Semantic Search
- Vector Embeddings
- Conversational AI
- Source Context Retrieval
